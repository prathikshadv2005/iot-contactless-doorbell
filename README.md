# 🚪 Contactless IoT Doorbell & Security System

## 📌 Overview
The Contactless IoT Doorbell & Security System uses a **PIR motion sensor and Raspberry Pi** to detect visitors near the door without physical contact. When motion is detected, a **buzzer rings as a doorbell alert** and an **LED indicates visitor presence**.

## 🧩 Components
- Raspberry Pi 4  
- PIR Motion Sensor  
- LED  
- Buzzer  
- 220Ω Resistor  
- Breadboard  
- Jumper Wires  

## ⚙️ Working
1. PIR sensor detects motion near the door.  
2. Raspberry Pi processes the signal.  
3. Buzzer rings like a doorbell.  
4. LED turns ON to indicate visitor detection.

## 🎯 Applications
- Smart Home Security  
- Contactless Doorbell System  
- Visitor Detection
